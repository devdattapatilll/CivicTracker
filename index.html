How It Works

CivicTrack uses YOLO detection and NLP classification to automatically identify and categorize civic issues

Workflow

1

Report

Capture a photo of the issue, describe the problem, and pin your GPS location. The entire process takes under a minute. No account required to browse.

2

AI Analysis

Three YOLO object detection models scan the uploaded image for potholes, garbage, or road cracks. The system automatically determines the issue category from the image. A separate TF-IDF + LinearSVC classifier categorizes text descriptions into matching categories.

3

Track and Resolve

Issues are stored in the database with real-time status tracking. Administrators can update issue status from Pending to In Progress to Resolved.

System Architecture

Frontend

Vercel (Static Hosting)

  • HTML5 + CSS3 single-page application
  • React 18 for dynamic components
  • Firebase SDK for auth and database
  • Responsive sidebar navigation

ML Backend

Render (Python / Flask)

  • 3 YOLO object detection models
  • TF-IDF + LinearSVC text classifier
  • REST API with 6 endpoints
  • Gunicorn production server

Database

Firebase (Google Cloud)

  • Authentication (Google + Email)
  • Firestore NoSQL document store
  • Cloud Storage for uploaded images
  • Real-time data synchronization
